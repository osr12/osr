OSR Car Charger ⚡ (Dark/Light) - Quick Start
============================================

1) XAMPP
--------
- شغّل Apache + MySQL من لوحة XAMPP.

2) مجلد المشروع
---------------
- انسخ المجلد "OSR_Car_Charger_UI" إلى: C:\xampp\htdocs\

3) قاعدة البيانات
-----------------
- افتح phpMyAdmin: http://localhost/phpmyadmin
- أنشئ قاعدة: osr_charger
- استورد الجداول من الملف osr_charger.sql

4) تشغيل الموقع
---------------
- افتح: http://localhost/OSR_Car_Charger_UI/
- أنشئ حساب جديد وسجّل الدخول.

5) إعادة تعيين كلمة المرور
--------------------------
- ملف forgot.php يرسل بريد عبر دالة mail() القياسية.
- يُفضَّل استخدام SMTP مع PHPMailer لتحسين التسليم.
- لتفعيل SMTP: ثبّت PHPMailer ثم عدّل forgot.php لإعداد SMTP.

6) تنبيهات 15 دقيقة
-------------------
- الواجهة تعرض تحذير بصري عندما يتبقى 15 دقيقة (إطار متقطع).
- لإرسال بريد/إشعار تلقائي يمكنك إنشاء مهمة مجدولة على ويندوز لتشغيل سكربت cron (غير مضمّن).

Have fun! ⚡
